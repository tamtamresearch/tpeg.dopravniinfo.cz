# Changelog

A high-level log of changes to the pilot service and to this site. For code-level history, see the [git log](https://github.com/tamtamresearch/tpeg.dopravniinfo.cz/commits/main).

## v0.1 — Pilot launched

- Pilot TPEG2 service goes live for evaluation.
- Two application formats published:
    - **TPEG2-TEC** — `x-format:cz-ndic_tpeg2-tec-v0.1`
    - **TPEG2-TFP** — `x-format:cz-ndic-tpeg2-tfp-v0.1`
- This site is launched as the public face of the pilot.
- Subscription channel: [tpeg@ceda.cz](mailto:tpeg@ceda.cz).
