NDIC HTTP Protocol for Sending All Changes on Occurrence¶

Identifier and version:

uri:

Protocol:

Specification:

1.1rev1

Updated:

2023-03-09

Publisher and author:

Publisher:

Národní Dopravní Informační Centrum

Author:

Jan Vlčinský, Petr Bureš

This document specifies the so-called PUSH protocol. The PUSH protocol enables the sending of messages containing information from the provider’s system to the recipient’s web service at the moment when they are updated. This protocol is used either to send complete data snapshots when the entire snapshot is updated or to send updates of individual records.”
