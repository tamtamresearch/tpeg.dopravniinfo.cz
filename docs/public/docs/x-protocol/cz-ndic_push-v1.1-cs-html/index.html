NDIC HTTP odesílání všech změn při výskytu¶

Identifikátor a verze:

uri:

cz-ndic_push-v1.1

Protokol:

1.1

Specifikace:

1.1rev1

Aktualizováno:

2023-03-09

Vydavatel a autor:

Vydavatel:

Národní Dopravní Informační Centrum

Autor:

Jan Vlčinský, Petr Bureš

Tento dokument specifikuje tzv. PUSH protokol. Protokol PUSH umožňuje zasílání zpráv s informacemi ze systému poskytovatele na webovou službu odběratele ve chvíli, kdy dojde k jejich aktualizaci. Tento protokol se užívá buď k zaslání ucelených snímků dat v okamžiku, kdy je celý snímek aktualizován, a nebo k zasílání aktualizací dílčích záznamů.
