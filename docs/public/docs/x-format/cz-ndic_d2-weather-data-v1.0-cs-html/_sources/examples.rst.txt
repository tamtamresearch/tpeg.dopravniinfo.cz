Příloha A: Příklady zpráv
=========================

meteo-data.xml - data o počasí naměřená jednou meteostanicí
===========================================================

Publikace obsahuje jeden záznam o jednom zařízení schopném několika různých měření (schopnosti).

Podrobnosti o zařízení (schopnosti a poloha) jsou definovány v publikaci typu `MeasurementSiteTablePublication`, zde nejsou podrobně popsány.

Zařízení je označeno odkazem na tabulku měřích míst `id="CZ_DRU_MST_WEA"` (s verzí) a odkazem na místo měření `id="12"` (s verzí).

Kořenový prvek a obálka publikace
---------------------------------

Kořenový element `d2LogicalModel` deklaruje, že se jedná o zprávu používající DATEX II.

Element `exchange` obsahuje identifikátor poskytovatele.

Element `payloadPublication` je obálkou pro přenos samotných informací, obsahující odkaz na tabulku měřících míst `measurementSiteTableReference` a jeden element `siteMeasurements`.

Atribut `xsi:type="MeasuredDataPublication"` v elementu `payloadPublication`
určuje, že tato obálka je typu Publikace měřených dat.

Element `feedType` uvádí typ publikace, uri schématu.

Element `publicationTime` uvádí čas vydání této publikace jako celku, což může být čas, pro který byly vypočteny všechny údaje o počasí (začátek platnosti).

Element `publicationCreator` identifikuje tvůrce publikace.

Dopravní veličiny z jednoho měřícího místa
------------------------------------------

Element `siteMeasurements` obsahuje jedno nebo více měření naměřených na konkrétním zařízení `measurementSiteReference`. `measurementSiteReference` je identifikován podle id a verze.

Element `measureTimeDefault` udává čas platnosti sady měření v rámci jednoho místa měření. Je také čas, kdy byly vytvořeny naměřené údaje o počasí (tj. konec měřící periody).

Následující element `measuredValue` (identifikovaný indexem) se může opakovat a popisuje měřené údaje o počasí (schopnosti) pro konkrétní měřicí místo.

Zařízení (meteostanice) umožňuje 9 různých měření (schopností). Jedná se o:

- srážky,
- podmínky povrchu vozovky (3x),
- teplota,
- vlhkost vzduchu,
- viditelnost a
- vítr.

Měření související s vlastnostmi povrchu vozovky je zde 3 krát (indexy 2-4). První definuje celkové charakteristiky a používá se ke zprostředkování stavu silnice. Zbývající dvě jsou měření související s jízdními pruhy (jedna pro každý jízdní pruh). Měření pro druhý jízdní pruh (index 4) je v chybovém stavu `measurementEquipmentFault`.

Vnořený prvek `basicData` obsahuje aktuální naměřenou hodnotu daného typu.

Časová platnost
^^^^^^^^^^^^^^^

Začátek platnosti měřených dat je stanoven povinným prvkem `measureTimeDefault`.
