========
Koncepty
========

Popisovaný formát umožňuje publikovat údaje o počasí (vlhkost, znečištění, srážky, povrchové podmínky vozovky, teplota, viditelnost a vítr) pro soubor předdefinovaných meteorologických stanic.

Typ publikace MeasuredDataPublication
=====================================

Formát vychází z publikace DATEX II typu `MeasuredDataPublication` [16157-5]_ a slouží k publikování dat o počasí pro sadu předdefinovaných měřicích zařízení.

.. literalinclude:: ../../samples/meteo-data.xml
   :language: xml
   :linenos:
   :lineno-match:
   :end-before: <measurementSiteTableReference

Údaje o počasí jsou poskytovány v prvku `measurementSiteReference` pro určité měřicí zařízení definované ve statické publikaci  `MeasurementSiteTablePublication`, která je odkazována pomocí jedinečného id a verze v prvku `measurementSiteTableReference`.

Prvek `measurementSiteReference` se sadou měření pro konkrétní měřicí místo se může objevit v `siteMeasurements` vícekrát (pro různé meteostanice).

Předdefinované místa měření
===========================

Schopnosti a poloha meteostanice se nemění. Meteostanice je proto v tomto profilu popsána odkazem (id) do předdefinované publikace tabulek měřicích míst definovaných v jiném dokumentu. Odkaz vede na instanci třídy `MeasurementSiteRecord` z balíčku` MeasurementSiteTablePublication`. Poloha a schopnosti zařízení proto nejsou popsány přímo v souboru.

Informace o zařízení (poloha a schopnosti) jsou určeny odkazem na soubor předdefinovaných zařízení. Pro vyhledání daného zařízení je nutné získat publikaci `MeasurementSiteTablePublication`, která se vztahuje k tomuto zdroji informací a nalézt v ní záznam stejného typu (podle `targetClass`, zde `MeasurementSiteRecord`) se odpovídajícím identifikátorem a verzí umístěný v rámci odpovídající tabulky měřících míst.

Plně kvalifikovaný identifikátor měřicího místa se skládá z:

- id a verze tabulky měřících míst a
- id a verze měřícího místa.

.. literalinclude:: ../../samples/meteo-data.xml
   :language: xml
   :linenos:
   :lineno-match:
   :start-after: </publicationCreator>
   :end-before: <!--precip_type + precipitation -->

Schopnosti meteostanice
=======================

Každá meteorologická stanice v tomto profilu je schopna provádět různé sady měření:

- vlhkost vzduchu,
- znečištění,
- srážky,
- podmínky povrchu vozovky,
- teplota,
- viditelnost a
- vítr.

Úplný seznam možných měření danou meteostanicí, nazývaných schopnosti, je stanoven v `MeasurementSiteTablePublication` pro každou stanici doslovně, jako výskyt záznamů `measurementSpecificCharacteristics` identifikovaných indexem. Pro odkazování na konkrétní schopnost (tj. vítr), použije se index schopnosti.

.. literalinclude:: ../../samples/meteo-data.xml
   :language: xml
   :linenos:
   :lineno-match:
   :start-after: <!--precip_type + precipitation -->
   :end-before: <!--road_temp 1 + freez_temp-->

Zveřejnění pouze aktuálně dostupných dat
========================================

Formát poskytuje informace o schopnostech se známým stavem. Publikována je každá známá hodnota (může být nula) či informace, že je daná schopnost (měření) chybové (např. neznámé). K publikaci hodnoty schopnosti v elementu `measuredValue` s danou hodnotou indexu, nedojde pouze tehdy není li možno hodnotu vypočíst (např. množství srážek v případě že neprší).

===============
Časová platnost
===============

Doba platnosti všech dat na naměřených jedním zařízením je dána povinným prvkem `measuremetnTimeDefault` vnořeným v `measurementSiteReference`. Každé místo měření musí tedy mít vlastní čas platnosti.
Dobou platnosti se rozumí konec měřící periody, kdy jsou vypočteny a poskytnuty údaje.

Žádná další specifikace platnosti není použita.

   :language: xml
   :linenos:

   :end-before: <!--precip_type + precipitation -->

====================
Odkazování na polohu
====================

Formát nepopisuje polohu, ten se očekává jako součást specifikace zařízení v publikaci `MeasurementSiteTablePublication` [16157-5]_.

Metody odkazování na polohu jsou mimo rozsah tohoto dokumentu a jsou popsány jinde.
