=========================
Příloha A: Příklady zpráv
=========================

restriction.xml - omezení vjezdu nákladních vozidel nad 3,5 tuny
================================================================

Publikace obsahuje právě jednu zprávu o uzavření pozemní komunikace pro vjezd nákladních vozidel s hmotností na 3,5 tuny.

.. literalinclude:: ../../samples/restriction.xml
   :language: xml

Kořenový element a obálka publikace
-----------------------------------

Kořenový element `d2LogicalModel` deklaruje, že se jedná o zprávu používající DATEX II.

Element `exchange` obsahuje identifikátor poskytovatele.

Element `payloadPublication` je obálkou pro přenos samotných informací,
obsahující především elementy `situation`, kterých může být mnoho (v našem případě je jeden).

Atribut `xsi:type="SituationPublication"` v elementu `payloadPublication`
určuje, že tato obálka je typu Publikace situace.

Element `feedType` uvádí typ publikace, uri schématu.

Element `publicationTime` uvádí čas vydání této publikace jako celku.

Element `publicationCreator` identifikuje tvůrce publikace.

Popis jedné konkrétní situace
------------------------------

Následující element `situation` se může opakovat a popisuje jednotlivé samostatné dopravní informace. Element `situationRecord` se v tomto případě vyskytuje v elementu `situation` právě jednou.

Element `situationVersionTime` uvádí čas vydání této verze situace jako celku.

Element `headerInformation` popisuje způsob nakládání s popisovanými informacemi.

Popis události (co se stalo)
^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Atribut `xsi:type="RoadOrCarriagewayOrLaneManagement"` elementu
`situationRecord` určuje typ zprávy, při pohledu do DATEX II modelu
lze ověřit, že jde o označení typu potomka třídy `OperatorActions`
v této dědičné posloupnosti `RoadOrCarriagewayOrLaneManagement` ->
`NetworkManagement` -> `OperatorActions` , používaných v tomto formátu.

`complianceOption` s hodnotou `mandatory` určuje, že jde o závazné omezení.

Element `forVehiclesWithCharacteristicsOf` upřesňuje hodnotou `lorry`
ve `vehicleType`, že se omezení týká nákladních vozidel a v
`grossWeightCharacteristic` pak to, že se týká vozidel těžších, než 3,5
tuny.

`roadOrCarriagewayOrLaneManagementType` hodnotou `roadClosed` vyjadřuje, že
pro uvedené typy vozidel je předmětná komunikace uzavřena.

Časová platnost
^^^^^^^^^^^^^^^

Časová platnost situace je určena ve strukturách `validity` odhadem časového intervalu platnosti hodnotami od `overallStartTime` do `overallEndTime`.

Poloha události
^^^^^^^^^^^^^^^

Poloha události je popsána v elementu `groupOfLocations` a to přímo
vnořeným popisem, není používáno odkazu na předem definované
polohy. Atribut `xsi:type="Linear"` uvádí, že jde o lineární typ polohy
a ten je zde vyjádřen třemi popisy. Nejprve pomocí lokačních tabulek v elementu `alertCLinear` typu `AlertCMethod2Linear`, poté pomocí obecného popisu polohy na silnici v elementu `linearWithinLinearElement` a následně popsán pomocí odkazů do sítě Global Network v elementu `globalNetworkLinear`.

Všechny použité metody popisu polohy vyjadřují tutéž polohu.

Další obsah
^^^^^^^^^^^

Mimo časovou platnost situace dokument uvádí čas vydání první verze `situationRecordCreationTime` a aktuální verze `situationRecordVersionTime` záznamu `situationRecord`.

Element `source` uvádí tvůrce této informace, tedy kdo tuto informaci vytvořil a odpovídá za její správnost.

Element `generalPublicComment` obsahuje obecný popis děje popsaného ve zprávě, většinou jen v češtině. Jedná se o volný popis situace jejím tvůrcem.
