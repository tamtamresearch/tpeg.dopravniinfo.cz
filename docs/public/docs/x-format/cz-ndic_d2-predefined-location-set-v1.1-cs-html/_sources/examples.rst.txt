predefined_location_set.xml - publikace se dvěma předdefinanými polohami
========================================================================

Publikace obsahuje dvě předdefinované polohy.

  .. literalinclude:: ../../samples/predefined_location_set.xml
   :language: xml

Kořenový element a obálka publikace
-----------------------------------

Kořenový element `d2LogicalModel` deklaruje, že se jedná o zprávu používající DATEX II.

Element `exchange` obsahuje identifikátor poskytovatele.

Element `payloadPublication` je obálkou pro přenos samotných informací,
obsahující především elementy `predefinedLocationContainer`, kterých je v publikaci tolik kolik je předdefinovaných poloh (v příkladu jsou pouze dvě).

Atribut `xsi:type="PredefinedLocationsPublication"` v elementu `payloadPublication`
určuje, že tato obálka je typu Publikace předdefinovaných poloh.

Element `feedType` uvádí typ publikace, uri schématu.

Element `publicationTime` uvádí čas vydání této publikace jako celku.

Element `publicationCreator` identifikuje tvůrce publikace.

Popis předdefinovaných poloh
---------------------------- 

Předem definovaná poloha události je popsána v elementu `predefinedLocationContainer` a to přímo vnořeným popisem. 

Publikace v příkladu popisuje dvě předdefinované polohy s identifikátory `id="dde51831-2f37-4f07-8429-286a9d08106c"` a `id="da0a5990-530c-426a-a6fd-ce88aeeb8aab"`.

Popis polohy (dde51831-2f37-4f07-8429-286a9d08106c)
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Element `predefinedLocationName` obsahuje název lokace.

Element `location` pak samotnou definici polohy. Atribut `xsi:type="Linear"` uvádí, že jde o lineární typ polohy.

Poloha je zde vyjádřena pouze popisem pomocí odkazů do sítě Global Network v elementu `globalNetworkLinear`.

Popis polohy (da0a5990-530c-426a-a6fd-ce88aeeb8aab)
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Element `predefinedLocationName` obsahuje název lokace.

Element `location` pak samotnou definici polohy. Atribut `xsi:type="Linear"` uvádí, že jde o lineární typ polohy.

Poloha je zde vyjádřena dvěma popisy. Nejprve pomocí lokačních tabulek v elementu `alertCLinear` typu `AlertCMethod2Linear`, poté pomocí odkazů do sítě Global Network v elementu `globalNetworkLinear`.

Všechny použité metody popisu polohy vyjadřují tutéž polohu.
