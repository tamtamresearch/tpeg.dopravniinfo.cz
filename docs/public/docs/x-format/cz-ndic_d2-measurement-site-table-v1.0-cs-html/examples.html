Kořenový element a obálka publikace¶

Kořenový element d2LogicalModel deklaruje, že se jedná o zprávu používající DATEX II.

Element exchange obsahuje identifikátor poskytovatele.

Element payloadPublication je obálkou pro přenos samotných informací, obsahující především elementy measurementSiteTable, a v nich vnořené elementy measurementSiteRecord kterých je v publikaci tolik kolik je předdefinovaných zařízení (v příkladu jsou pouze dvě).

Atribut xsi:type=“MeasurementSiteTablePublication“ v elementu payloadPublication určuje, že tato obálka je typu Publikace předdefinovaných zařízení.

Element feedType uvádí typ publikace, uri schématu.

Element publicationTime uvádí čas vydání této publikace jako celku.

Element publicationCreator identifikuje tvůrce publikace.

Popis předdefinovaných zařízení¶

Předdefinované zařízení je popsáno v tabulkách měřicích míst, které jsou jednoznačně identifikovány podle id a verze v příkladu id=“CZ_DRU_MST_DET“ pro tabulku detektorů a id=“CZ_DRU_MST_WEA“ pro tabulku meteorologických stanic.

Každá tabulka v příkladu obsahuje pouze jeden prvek popisující příslušné schopnosti zařízení. Identifikace zařízení je jedinečná v dané tabulce zařízení. Detektor s id=“1307“ a id=“12“ pro meteorologickou stanici.

Popis detektoru (CZ_DRU_MST_DET, 1307)¶

Element measurementSiteRecord obsahuje textový popis zařízení.

Zařízení (detektor provozu) má 21 definovaných schopností. Jedná se o kategorizované verze tří dopravních veličin:

  • dopravní tok,

  • dopravní koncentrace,

  • rychlost provozu.

První tři schopnosti measurementSpecificCharacteristics s indexy 1-3 popisují tok, obsazenost a rychlost pro všechna vozidla. Zbývající schopnosti popisují tyto hodnoty provozu kategorizované podle typu vozidla.

Např. schopnosti s indexy 4-9 popisují tok provozu pro:

  • auto

  • nákladní automobil

  • přívěs

  • dodávka

  • autobus

  • motocykl

Schopnosti measurementSpecificCharacteristics s indexy 10-15 popisují kategorizovanou obsazenost a kategorizovanou rychlost popisují schopnosti s indexy 16-21.

Element measurementSiteLocation typu xsi:type=“Point“ obsahuje jednotlivé metody popisu místa, v našem případě pouze jeden popis pomocí odkazů na Global Network v elementu pointWithinLinearGNElement.

Popis meteostanice (CZ_DRU_MST_WEA, 12)¶

Prvek measurementSiteRecord obsahuje textový popis zařízení.

Zařízení (meteorologická stanice) má 8 definovaných schopností:

  • srážky,

  • podmínky povrchu vozovky (3x),

  • teplota,

  • vlhkost vzduchu,

  • viditelnost a

  • vítr.

podmínky na povrchu vozovky jsou uvedeny 3 krát (indexy 2-4). První definuje celkové vlastnosti a používá se ke zprostředkování informace o stavu silnice. Zbývající dvě jsou měření vztažené vůči konkrétní poloze na silnici (jeden pro každý jízdní pruh).

Element measurementSiteLocation typu ‚xsi:type=“Point“` obsahuje jednotlivé metody popisu místa, v našem případě pouze jeden popis pomocí odkazů na Global Network v elementu pointWithinLinearGNElement.
