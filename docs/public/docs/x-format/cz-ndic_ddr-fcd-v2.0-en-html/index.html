Author:

Jan Vlčinský, Petr Bureš

This document describes a data format, the NDIC Data Distribution Interface (DDR), used to provide information from floating vehicles (travel times, speed, traffic conditions and queue length, number of vehicles, types) at a predefined set of locations. Separated by vehicle type (car, truck). The data format is not backward compatible with the original format, introduces a hierarchical structure to the elements, renames them and modifies the meaning.
