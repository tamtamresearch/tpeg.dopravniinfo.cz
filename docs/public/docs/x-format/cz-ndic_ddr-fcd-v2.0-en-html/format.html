Appendix B: Representative XML Schema: RELAX NG¶

The RELAX NG schema in a compact format is used to describe the XML structure. At the same time, the traditional W3C XML Schema 1.0, created by conversion from RELAX NG, is also available.

RELAX NG Compact Syntax Schema (RNC)¶

RNC schema describes expected structures of XML documents.

Root element is introduced by start =

All other <SomeName> = elements define named patterns.

In our schema there are following sections:

  • prolog (namespace declaration)

  • includes (not used here)

  • root element definition (start =)

  • patterns for elements

  • patterns for attributes

  • patterns for data types

Comments ## specify more details about meaning of particular parts (in Czech language).

schema-cs.rnc - root of the schema¶

default namespace = "x-format:cz-ndic_ddr-fcd-v2.0"

start =

## kořenový element publikace
element DOC {

  ## čas vytvoření dokumentu
  attribute created { xsd:dateTime },

  ## kontextové informace k publikaci
  element INF {

    ## informace o souvisejících datových sadách
    element DAT {

      ## DATEX II sada předdefinovaných míst
      element D2PLS {

        ## uri zdroje (implicitně nebo explicitně zahrnuje i verzi)
        attribute uri { xsd:NMTOKEN }
      }
    }
  },

  ## kontejner pro informace z jednotlivých segmentů
  element FCD {

    SEGMENT+
  }
}

SEGMENT = 
  ## informace pro jeden segment
  element SEGMENT {

    ## identifikátor segmentu ze sady předdefinovaných míst
    attribute id { xsd:NCName },

    ## verze popisu polohy segmentu s daným id
    attribute version { xsd:integer },

    ## čas, pro který jsou data vypočtena [dateTime]
    ##
    ## poslední okamžik kdy byly pro segment vypočteny (aktualizovány) hodnoty. 
    ## K aktualizaci dochází pravidelně v intervalu 1 minuty, kdy se vyhodnocují trasy všech vozidel
	## na sledované síti a jimi projeté úseky a vyhodnocují se hodnoty k takto projetým úsekům.
	## K poslední aktualizaci stavu segmentu dojde, když jej opustí poslední vozidlo.
	## 
    ## Segmenty jejichž poslední aktualizace proběhla před více než 5 minutami nejsou do celkového přehledu poskytnuty (nejsou aktuální). 
    attribute last_updated { xsd:dateTime },

    ALL,

    CAR?,

    TRUCK?
  }+

ALL = 

  ## hodnoty bez rozlišení typů vozidel (sdružený dopravní proud)
  element ALL {

    ## počet vozidel použitých pro výpočet hodnot [-] 
    attribute count { xsd:integer },

    ## aktuální rychlost dopravního proudu [km/h]
    ##
    ## průměrná rychlost vozidel které se na segmentu vyskytují.
    attribute speed { xsd:integer },

    ## aktuální dojezdová doba [s]
    ##
    ## informace o aktuálním čase průjezdu vypočtená z „délka segmentu/průměrná rychlost na segmentu“
    attribute travel_time { xsd:integer },

    ## zpoždění [s]
    ##
    ## informace o zdržení vypočtená z jako „čas průjezdu volného segmentu - aktuální čas průjezdu“
    attribute delay { xsd:integer },

    ## stupeň dopravy dle zvyklostí v ČR [-] 
    ##
    ## https://cs.wikipedia.org/wiki/Stupe%C5%88_provozu; 
    ## Zde vypočítané z % rozdílu aktuální rychlosti dopravního 
    ## proudu a rychlosti dopravního proudu na volném segmentu.
    attribute traffic_level { 

    ## stupeň 1: provoz pouze jednotlivých vozidel, jízda je plynulá
    ## rozmezí proc. rozdílu rychlostí < 85 %, 100 % )
      "1" |

    ## stupeň 2: provoz malých skupinek vozidel, jízda je plynulá, odbavování na křižovatkách bez problémů
    ## rozmezí proc. rozdílu rychlostí < 75 %, 85 % )
      "2" |

    ## stupeň 3: tvoří se proudy vozidel, provoz je plynulý, avšak rychlost nižší než maximální povolená
    ## rozmezí proc. rozdílu rychlostí < 50 %, 75 % )
      "3" |

    ## stupeň 4: tvoří se kolony vozidel, provoz není plynulý, průměrná rychlost je výrazně snížená, průjezd křižovatkami je narušen
    ## rozmezí proc. rozdílu rychlostí < 20 %, 50 % )
      "4" |

    ## stupeň 5: dopravní kolaps – vozidla na komunikacích stojí nebo v kolonách jen velmi pomalu popojíždějí, průměrná rychlost je velmi malá
    ## rozmezí proc. rozdílu rychlostí < 0 %, 20 % )
      "5"
  },

    ## rychlost volného průjezdu [km/h]
    ##
    ## historická hodnota rychlosti volného průjezdu 
    ## agregovaná přes všechna období.
    attribute freeflow_speed { xsd:integer },

    ## doba volného průjezdu [s]
    ##
    ## historická hodnota vypočtena jako délka segmentu/rychlost volného průjezdu
    attribute freeflow_time { xsd:integer },

    ## spolehlivost dat, která charakterizuje kvalitu datového vzorku [%]
    ##
    ## Nabývá hodnot <0-1> kdy 1 znamená 100% spolehlivá data.
    ## Závislá pouze na počtu vozidel na segmentu, základní hodnota 50 % pro 1 vozidlo 
    ## na segmentu, s více vozidly na segmentu asymptoticky stoupá ke 100 %
    attribute reliability { xsd:decimal },

    CONGESTION?
  }

CAR = 

  ## hodnoty pro osobní vozidla
  element CAR { 

    ## počet vozidel použitých pro výpočet hodnot  [-] 
    attribute count { xsd:integer },

    ## aktuální rychlost dopravního proudu [km/h]
    ##
    ## průměrná rychlost vozidel které se na segmentu vyskytují.
    attribute speed { xsd:integer },

    ## rychlost volného průjezdu [km/h]
    ##
    ## historická hodnota rychlosti volného průjezdu 
    ## agregovaná přes všechna období.
    attribute freeflow_speed { xsd:integer },

    CONGESTION?
  }

TRUCK = 

  ## hodnoty pro nákladní vozidla
  element TRUCK {
    ## počet nákladních použitých pro výpočet hodnot
    attribute count { xsd:integer },

    ## aktuální rychlost dopravního proudu [km/h]
    ##
    ## průměrná rychlost vozidel které se na segmentu vyskytují.
    attribute speed { xsd:integer },

    ## rychlost volného průjezdu [km/h]
    ##
    ## historická hodnota rychlosti volného průjezdu 
    ## agregovaná přes všechna období.
    attribute freeflow_speed { xsd:integer },

    CONGESTION?
  }

CONGESTION = 

  ## Popis konkrétní kolony
  ##
  ## Segmenty jsou interně děleny na menší části (úseky GN).
  ## Na kterékoliv z těchto interních části vzniká kolona,
  ## pokud na ní rychlost klesne pod 25% rychlosti volného průjezdu.
  ##
  ## V současné chvíli se nerozlišuje je-li kolona generována osobními
  ## či nákladními vozidly, do výpočtu tedy vstupují všechny úseky segmentu 
  ## na kterých vozidlo či vozidla zpomalila na 30 % rychlosti volného průjezdu.
  element CONGESTION {

    ## začátek kolony [m] měřeno od počátku segmentu ve směru dopravního proudu
    ##
    ## v případě více rozdělených kolon na segmentu, například kolona detekovaná 
    ## na 1. a 3. úseku GN atribut from udává pouze informaci o 1. koloně 
    attribute from { xsd:integer },

    ## agregovaná délka kolony na všech úsecích segmentu [m]
    ##
    ## resp. délka úseku GN na kterém byla kolona detekována. 
    ## v případě více rozdělených kolon na segmentu atribut length 
    ## udává sloučenou délku všech úseků segmentu s detekovanou kolonou
    attribute length { xsd:integer }

  }
