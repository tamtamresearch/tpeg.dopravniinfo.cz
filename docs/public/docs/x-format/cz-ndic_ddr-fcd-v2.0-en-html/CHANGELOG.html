Changelog¶

version 2.0

  • Rewrite of CONGESTION tag, where formerly was outside of ALL, CAr and TRUCK tags, now is inside allowing to have different values for different types of traffic flows.

version 1.1.0

  • Complete rewrite of the specification. Formerly based on a tmc tag holding in as attributes all values

  • the information is now structured into SEGMENTs, with id, version and update time and information contained within the tag is furthermore separated in to tags - ALL with traffic parameters for non-divided traffic flow - CAR and TRUCK for information about separated traffic flows - CONGESTION for congestion length information

version 1.0.0

  • Initial format and documentation
