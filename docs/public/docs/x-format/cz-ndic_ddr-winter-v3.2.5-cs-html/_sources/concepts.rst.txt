========
Koncepty
========

Tato kapitola popisuje základní koncepce, používané v tomto formátu. Detailní popis viz definice ve schématech.

Tato kapitola uvádí krátké ukázky ze skutečných XML dokumentů. Více ukázek viz :ref:`examples`.


Části XML dokumentu
===================

XML dokument staví na následujících elementech a částech:
- DOC (kořen dokumentu)
- INF (metadata)
- MJD (kontejner pro zprávy)
- MSG (zprávy)

Kořen dokumentu (`DOC`)
-----------------------

Element `DOC` představuje jednotlivou publikaci (jednu instanci přenášených dat).

Atribut `id` uvádí identitu dané publikace jako celku.

.. literalinclude:: ../../samples/new-snow-cover.xml
   :language: xml
   :linenos:
   :end-before: <MSG

Metadata (`INF`)
----------------

Element `INF` popisuje meta-informace o celé publikaci:

- referenci na použitou GDF 4.0 mapovou sadu (vydavatel a verze)
- referenci na číselník organizačních jednotek ČR UIR-ADR

.. literalinclude:: ../../samples/new-snow-cover.xml
   :language: xml
   :linenos:
   :start-after: extended
   :end-before: <MSG

Kontejner zpráv (`MJD`)
-----------------------

Jednotlivé zprávy jsou umístěny do jednoho společného rodičovského elementu `MJD`. Tento element může obsahovat jednu nebo více zpráv.

Zpráva (`MSG`)
--------------

Jednotlivé zprávy jsou reprezentovány elementem `MSG`.

Samotná zpráva má následující části:

- platnost v čase `MTIME`
- textový popis celé události `MTXT`
- charakteristika počasí `MEVT/WCOND`
- charakteristika sjízdnosti `MEVT/MTNCOND`
- odkaz na polohu `WDEST` (*kde*) v číselníku zimních zpravodajských oblastí.
- odkaz na region v UIR-ADR `MDST`

.. literalinclude:: ../../samples/new-snow-cover.xml
   :language: xml
   :linenos:
   :lineno-match:
   :start-after: <MJD
   :end-before: </MJD>

Identita zprávy a verze
^^^^^^^^^^^^^^^^^^^^^^^

Samotná zpráva má vlastní unikátní identifikátor `id` a verzi. Verze začínají číslem `1` a s každou aktualizací je verze zvýšena o jedničku.

Platnost zprávy v čase je specifikována jednoduchým intervalem platnosti od - do, ale z praktického hlediska je relevantní pouze okamžik “od”.

.. literalinclude:: ../../samples/new-snow-cover.xml
   :language: xml
   :linenos:
   :lineno-match:
   :start-after: <MJD
   :end-before: <MTXT

Charakteristika počasí
^^^^^^^^^^^^^^^^^^^^^^

Element `MEVT/WCOND` obsahuje sadu meteorologických parametr°u (teplota, srážky, vítr, viditelnost atp.).

   :linenos:

^^^^^^^^^^^^^^^^

Element `MEVT/MTNCOND` obsahuje parametry, popisující sjízdnost.

.. literalinclude:: ../../samples/new-snow-cover.xml

   :linenos:

Popis polohy
^^^^^^^^^^^^

DDR (Zimní sjízdnost) používá odkazy na zpravodajské oblasti, ze kterých jsou informace poskytovány.

Jsou použity následující metody odkazování na polohu:

- Odkaz do číselníků zpravodajských oblastí (viz :ref:`regioncodes`)
- Souřadnice středu zpravodajské oblasti.
- Odkaz na organizační oblast v číselníku UIR-ADR.

.. literalinclude:: ../../samples/new-snow-cover.xml
   :language: xml
   :linenos:
   :lineno-match:
   :start-after: </MEVT>
   :end-before: </MSG>
